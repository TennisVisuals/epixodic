### 3. Legacy Generator Redirects ✅

Legacy generators now redirect to UMO v4 for backward compatibility:

#### sampleMatch.ts
```typescript
/**
 * @deprecated Use generateSampleMatchUpV4() instead
 */
export function generateSampleMatch(): UMOv4Episode[] {
  return generateSampleMatchUpV4().episodes;
}
```

#### sampleGames.ts
```typescript
/**
 * @deprecated Use generateMultipleGamesV4() instead

```
